# Chess Challenge

A simple chess movement simulator made in C for a college programming challenge, based on the original challenge repository:

- [desafio-xadrez-sabrigs](https://github.com/Cursos-TI/desafio-xadrez-sabrigs)

This project was created to practice:

- Recursion
- Loops (`for`, `do-while`)
- Functions
- Switch cases
- Code organization in C

## Pieces

The program supports movement simulation for:

- Bishop
- Tower
- Queen
- Horse (Knight)

Each piece prints its movement direction step-by-step in the terminal.
